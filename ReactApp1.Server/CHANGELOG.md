Ten plik wyjaśnia, jak Visual Studio utworzyło projekt.

Następujące kroki zostały użyte do wygenerowania tego projektu:
- Utwórz nowy projekt ASP\.NET Core Web API.
- Zaktualizuj plik projektu, aby dodać odwołanie do projektu frontonu i ustawić właściwości SPA.
- Zaktualizuj `launchSettings.json`, aby zarejestrować serwer proxy SPA jako zestaw startowy.
- Dodaj projekt do listy projektów startowych.
- Zapisz ten plik.
